### Vue 3.x



​	**通过 `vite` 搭建的 `vue 3.x` 项目**

​	**根据 `ts` 构建脚本模块**

​    **使用 `sass` 对`css` 进行预编译**

​    **vue：`createApp` `ref` `reactive` `readonly` `watchEffect` `computed`**

​    **vue-router: `useLink`  `useRoute` `useRouter` `parseQuery` `stringifyQuery`**

​    **vuex: `useStore`**
